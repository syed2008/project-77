Project on console

Visual Studio Code is a platform on which we can code and make websites. To set it up, you have to go on code.visualstudio.com and download it from there.

Console screen is a screen on which we can print different things while coding a website. To open a console screen on a website, you can either right-click or you can use the shortcut, ctrl + shift + i. Then, you have to click on console and you will have the console screen. To print something on the console screen, you can use the console.log command in js.
